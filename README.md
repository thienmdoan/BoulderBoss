# BoulderBoss
A VR application for rock climbers to check weather and enjoy simulated climbing experiences.

Technologies:
JavaScript
ES6 / ES2015
Babel
AJAX
Node.js
Express.js
RESTful API
Three.js
A-frame

To start:
1. git clone "repository".
2. npm install
3. input Open Weather API key to the API call in index.js
4. npm run dev
5. open on localhost:3000
